# 更新日志

# Ver 1.1.0 - 2024.6.29

  • 修复软件狗失效问题
  • 功能启动速度优化
  • 修复启动器飞时达冲突误报问题，修复部分已知插件冲突

  • 修复导出属性表图层名过长报错问题
  • 修复三维多段线转为多段线问题
  • 修复图层锁定时字段管理器刷新问题

# Ver 1.0.2.0 - 2024.4.22

  • 修复 字段计算器 部分函数无效问题

  • 检测并提醒和飞时达的冲突
  • 修复 Ribbon 菜单部分命令失效问题
  • 优化 导出CSV ，支持导出点 XYZ 值
  • 多段线折点转点，支持输出为CSV
  • 修复 KML 图层名称异常字符问题
  • 优化 栅格处理工具，支持波段设置
  • 优化 影像裁剪，支持多部件，支持洞
  • 添加 批量输出栅格属性 工具
  • 修复选目录无法获取目录内文件问题
  • 修复空间选择报错问题，默认启用多线程，效率优化
  • 更新部分图源，通过 图源管理 - 默认图源 添加（为避免重复，添加前可先全选删除已有图源）
  • 修复部分登陆出错问题
