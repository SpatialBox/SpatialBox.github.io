# 简介

SpatialBox 是一款基于 AutoCAD 的 GIS 扩展插件，围绕数据交互、属性管理、空间分析、栅格扩展等方面，系统性增强 AutoCAD 的地理数据处理能力。

# 功能模块

  • 数据交互：支持 SHP GDB KML JSON Excel SQLITE等数据的加载和导出。
  • 空间分析：支持拓扑验证，拓扑检查，并可进行空间连接，空间选择等。
  • 栅格扩展：支持加载大影像、光栅图像，在线地图浏览，下载，支持自定义图源。
  • 属性管理：完善的字段和属性支持、强大的字段计算器、属性选择，不依赖外部数据库。
  • 编辑工具：提供常规的一些点、线、面、块、文字等编辑功能。
  • 标注注记：提供字段标注，字段注记，长度标注等功能，支持动态标注。
  • 通用工具：系列工具的集合，诸如裁剪、批量打印输出、批量合并拆分等。

所有功能均基于CAD原生数据结构，不使用自定义实体，不使用外部数据库，文件共享更轻松。

# 安装卸载

软件提供绿色自解压安装包下载open in new window，默认安装目录为 D:\SpatialBox，要卸载删除该目录即可。

安装目录下的 user 文件夹可能有您自己保存的数据，删除前请注意检查备份。

为避免权限不足引发的错误，建议使用默认路径安装。

# 启用停用

安装后显示 SpatialBox启动器 界面，可启用、停用插件，可快速打开相应版本的 CAD。

当启动器左下角出现版本更新提示时，点击可查看和下载新版本。

某些精简版 CAD 可能移除了一些关键模块，导致插件无法正常运行，请注意安装完整版 CAD。

尽量使用2016及之后版本的CAD，运行更稳定流畅。
